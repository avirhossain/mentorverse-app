# **App Name**: MentorVerse

## Core Features:

- Admin Mentor Profile Creation: Admin creates detailed mentor profiles with required information, available timeslots, and session costs.
- Mentor Profile Display: Mentees can browse and view mentor profiles with available sessions.
- Session Booking: Mentees can purchase and book sessions with mentors through the app.
- Admin Session Offerings: Admin creates and offers unique sessions that mentees can book.
- Personalized Mentor Recommendations: AI analyzes mentee profiles and session history to recommend relevant mentors and unique sessions. The LLM acts as a tool, deciding when the fit is strong enough to incorporate recommendations.
- Mentorship goal: Allow the mentee to state their mentorship goal

## Style Guidelines:

- Primary color: Indigo (#4B0082) for trust and professionalism.
- Background color: Light gray (#EEEEEE) for a clean, modern look.
- Accent color: Teal (#A020F0) to highlight key actions and information, complementing the indigo.
- Body and headline font: 'Inter', a sans-serif font for clarity and modern feel.
- Use clean and professional icons to represent mentor categories and session types.
- Card-based layout for mentor profiles and sessions for easy browsing. This is a mobile first design.
- Subtle transitions and animations when navigating between mentor profiles and booking sessions.
